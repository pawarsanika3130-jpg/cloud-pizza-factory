# 🍕 Cloud Pizza Factory

A beginner-friendly DevOps project demonstrating Docker and Kubernetes concepts using a simple Python Flask application.

---

## 🚀 Project Overview

Cloud Pizza Factory simulates a real-world cloud-native application.

The project demonstrates:

- Docker Containerization
- Docker Images
- Kubernetes Deployments
- Pod Management
- Self-Healing
- Horizontal Scaling
